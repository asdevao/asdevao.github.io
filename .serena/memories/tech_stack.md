# Tech Stack

- **Static Site Generator**: Hugo (latest version)
- **Theme**: LoveIt (from GitHub)
- **Content**: Markdown files in `content/` directory
- **Styling**: CSS (from theme and custom)
- **JavaScript**: Vanilla JS for interactivity (language toggle, particles, archive views)
- **Libraries**: 
  - Mermaid for diagrams
  - ECharts for charts
  - TypeIt for typing animations
  - Mapbox for maps (if used)
- **Deployment**: GitHub Pages (baseurl: https://asdevao.github.io)
- **Version Control**: Git
- **Platform**: Windows development environment
