# Commands to Run After Completing Tasks

## Build Verification
- `hugo`: Build the site to verify no template errors or broken links
- Check for build errors in console output

## Local Testing
- `hugo server`: Start local server to visually inspect changes
- Test all modified pages for proper rendering

## Git Status Check
- `git status`: Review changes before committing
- `git diff`: Inspect modifications

## Deployment Preparation
- Ensure `public/` directory is up-to-date after build
- Test navigation and functionality on different devices/browsers

## Performance Check
- Verify page load times and asset loading
- Check for any console errors in browser dev tools
